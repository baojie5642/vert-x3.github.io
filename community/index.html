Community

Vert.x is an independent project and includes contributions from many individuals and organisations.

Get Help

User Group

A Google Group to discuss all issues related to using Vert.x.

Developer Group

A group for Vert.x core developers and contributors.

IRC channel

Wiki

The Vert.x wiki contains useful information such as a How to contribute guide and the Roadmap.

Issues

Please file issues related to the core Vert.x project in Eclipse Bugzilla. File issues related to non-core Vert.x projects in the appropriate GitHub issues.
